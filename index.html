Preprocessor Directives

Preprocessor directives in C are special instructions that are processed before the actual compilation of code begins. They begin with the '#' symbol and are used to include header files, define constants, and conditionally compile code. Common directives include #include for including libraries, #define for defining constants, and conditional directives like #ifdef, #ifndef, and #endif. These directives help in managing large codebases efficiently and make code reusable and modular. Additionally, macros created using #define can improve code readability and maintainability. Conditional compilation is useful for writing platform-independent code or including debugging statements only when necessary.

Tokens in C

Tokens are the smallest individual units in a C program that are meaningful to the compiler. The different types of tokens in C include:

  • Keywords: Reserved words like int, return, if, else.
  • Identifiers: Names given to variables, functions, arrays, etc.
  • Constants: Fixed values like 100, 3.14, 'A', etc.
  • String literals: Sequence of characters enclosed in double quotes, e.g., "Hello World".
  • Operators: Symbols used for computations, e.g., +, -, *, /, ==.
  • Special symbols: Characters like {}, [], (), ;, #, etc., that have syntactical meaning.

Understanding tokens is critical for parsing and writing syntactically correct C programs.

Formatted I/O Functions

Formatted input/output functions provide flexible ways to read from and write to standard input/output. The printf() function allows displaying data with control over formatting, such as number of decimal places, alignment, and more. Similarly, scanf() reads formatted input from the user.

int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("You entered: %d\n", age);

You can also format floats, strings, and characters using format specifiers like %f, %s, and %c. These functions are essential for interactive programs and user interfaces in console-based applications.

Unformatted I/O Functions

Unformatted I/O functions deal with character-level and string-level operations. Unlike formatted functions, they do not use format specifiers. Examples include:

  • getchar() – reads a single character from standard input.
  • putchar() – writes a single character to standard output.
  • gets() – reads a string from input (deprecated, use with caution).
  • puts() – writes a string followed by a newline.

These functions are simpler and faster but do not provide control over formatting. They're commonly used for reading and displaying quick messages or prompts.

Control Flow

Control flow in C determines the sequence of execution of statements in a program. It includes:

  • Decision-making: if, if-else, nested if, switch-case.
  • Loops: for, while, do-while for repetitive tasks.
  • Branching: break, continue, and goto for altering flow.

These constructs provide decision logic, iteration control, and branching capability to handle complex logical flows in a program efficiently.

Arrays

An array is a collection of elements of the same type stored in contiguous memory. It can be one-dimensional or multi-dimensional. You access elements using an index:

int arr[5] = {1, 2, 3, 4, 5};
printf("%d", arr[2]); // outputs 3

Arrays simplify the management of lists and tables of data and are widely used in sorting, searching, and storing large datasets in memory-efficient ways.

Array Operations

Common operations on arrays include:

  • Traversal: Using loops to access each element.
  • Insertion: Adding an element at a given position.
  • Deletion: Removing an element and shifting the rest.
  • Searching: Linear or binary search for finding elements.
  • Sorting: Bubble sort, selection sort, quicksort, etc.

Understanding these operations is key to manipulating and analyzing data stored in arrays.

Transpose

Transposing a matrix means flipping it over its diagonal, switching the row and column indices of each element. For example, if A is a 3x2 matrix, its transpose AT will be a 2x3 matrix.

for (i = 0; i < row; i++) {
  for (j = 0; j < col; j++) {
    transpose[j][i] = matrix[i][j];
  }
}

This operation is commonly used in scientific computing, graphics, and solving systems of equations.

Pattern Printing

Pattern printing is a great way to learn nested loops. You can print a variety of shapes like pyramids, triangles, diamonds using characters or numbers.

// Star triangle
for (i = 1; i <= n; i++) {
  for (j = 1; j <= i; j++) {
    printf("*");
  }
  printf("\n");
}

These problems enhance logical thinking and reinforce concepts of loop nesting and flow control.

String Handling Functions

C provides several built-in functions in <string.h> for string manipulation:

  • strlen() – Returns the length of a string.
  • strcpy() – Copies one string to another.
  • strcat() – Concatenates two strings.
  • strcmp() – Compares two strings.

String handling in C requires careful memory management since strings are null-terminated character arrays. These functions simplify operations like input validation, formatting, and file processing.
